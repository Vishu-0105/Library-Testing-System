This project focuses on testing the frontend of a Library Management System. It validates user interface components such as book listing, search functionality, issue and return actions to ensure correct behavior, proper input handling, and smooth user interaction.

Project Scope
Frontend testing only
No backend or database integration
Partial implementation for academic purposes

Features Tested
Display of available books
Search functionality
Issue book button behavior
Return book button behavior
Form validation and UI responses

Technologies Used
HTML
CSS
JavaScript
Testing Framework (e.g., Jest / Cypress / Selenium)
